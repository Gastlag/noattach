noatta.ch

Instant Temporary File Sharing

noatta.ch is made by Astro. The code is open source on Github: github.com/astro/noattach

How does it work?

  1. You get a random link that you can share privately with your friends.
  2. Browsing to that URL establishes a real-time connection.
  3. You instantly see what others share.
  4. When you add a new file, its file name and size will be broadcasted over the noatta.ch server to any other people visiting your URL.
  5. Downloading is a simple HTTP request to the server.
  6. The server will then individually tell the browser that shares the file to upload a chunk of it.
  7. After passing it to the downloader, it will continuously ask the sharer to pass the next chunk until end of file.

Why?

There are dropboxen and share hosters, but often, you'll want to share files only once, temporarily, with a limited set of recipients. That's what noatta.ch is made for: just point your browser at http://noatta.ch/ and pass the new URL to the people you want to share with.

Browser support?

On-demand sharing requires the File API.
