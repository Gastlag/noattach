Why?

There are dropboxen and share hosters, but often, you'll want to share files only once, temporarily, with a limited set of recipients. That's what noatta.ch is made for: just point your browser at http://noatta.ch/ and pass the the new URL to the people you want to share with.

How?

The server side runs on Node.js, together with Connect as the HTTP middleware. Communication is leveraged by Socket.io, but uploads are fairly normal XMLHttpRequests (AJAX) using the latest features (sendAsBinary & upload events).

Browser support?

On-demand sharing requires the File API.
